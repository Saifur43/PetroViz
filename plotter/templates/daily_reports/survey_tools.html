Instructions
Upload Survey File:
  • Select the well for which you want to upload survey data
  • Upload a .txt file containing survey station data
  • The file should contain MD, TVD, Northing, and Easting columns
  • Survey data will be imported and stored for depth conversions
Depth Conversion:
  • Select the well with survey data
  • Enter either MD or TVD value
  • The system will calculate the corresponding depth
  • Conversion uses interpolated survey station data
Populate Prognosis MD Depths:
  • Select a well that has survey data uploaded
  • All WellPrognosis entries for that well will be updated
  • TVD depths (planned_depth_start/end) will be converted to MD
  • The MD fields (md_depth_start/end) will be populated
